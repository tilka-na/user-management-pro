# user-management-pro
![User Management App](src/page.jpg)

Cette application permet de gérer des utilisateurs avec les fonctionnalités suivantes :  

- Ajouter un utilisateur (nom, email, âge)  
- Modifier un utilisateur existant  
- Supprimer un utilisateur  
- Afficher la liste des utilisateurs  
- Sauvegarde des données dans le `localStorage` du navigateur

Sur cmd
-git clone https://github.com/ton-nom-dutilisateur/user-management-pro.git
-cd user-management-pro
-python -m http.server 5500
Puis ouvrir votre navigateur et accéder à l’adresse :http://localhost:5500/
ou lancer le sur github page on specifiant branch (main), folder (/root) puis save.
